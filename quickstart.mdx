This guide walks you through downloading your first media file using VidArchive. By the end, you'll understand the basic workflow and be ready to explore more advanced features.

<Note>
  Make sure you have VidArchive installed. If not, check out the [installation guide](/installation) first.
</Note>

## Download your first media

<Steps>
  <Step title="Launch VidArchive">
    Open VidArchive from your applications menu or desktop shortcut.
    
    On first launch, VidArchive will automatically check for yt-dlp and ffmpeg. If they're not found, you'll be prompted to download them - just click **Download** and wait for the process to complete.
  </Step>

  <Step title="Enter a media URL">
    Navigate to the **Home** page using the sidebar. In the URL input field at the top, paste or type a media URL from any supported platform (YouTube, Twitter, Instagram, Reddit, TikTok, etc.).
    
    Click **Fetch** or press Enter to retrieve the media information.
    
    <Info>
      VidArchive automatically validates the URL and checks if the source is supported. If the URL is invalid or unsupported, you'll see an error message.
    </Info>
  </Step>

  <Step title="Review media information">
    Once the media information is fetched, you'll see:
    
    - **Preview thumbnail** - A visual preview of the media
    - **Title and uploader** - Basic metadata about the media
    - **Duration** - Total length of the media
    - **Format selector** - Pre-selected best quality format
    - **Details panel** - Additional information like upload date, view count, description
  </Step>

  <Step title="Select your format">
    Click the **Select format** dropdown to see the pre-selected format. If you want to choose a different format:
    
    1. Click **All formats** to open the format selection modal
    2. Browse through available formats organized by type (video+audio, video-only, audio-only)
    3. Each format shows resolution, codec, file size, and quality information
    4. Select your preferred format and click **Confirm**
    
    <Tip>
      For most use cases, the default "Best quality (video+audio)" selection works perfectly. It automatically picks the highest quality format with both video and audio.
    </Tip>
  </Step>

  <Step title="Configure download options (optional)">
    Expand the **Extra options** section to customize your download:
    
    **Embed options** - Add metadata to your file
    - Thumbnail as cover art
    - Chapters for navigation
    - Subtitles burned into video
    - Metadata fields (title, artist, album, etc.)
    
    **Write additional files**
    - Video description as text file
    - Comments/live chat
    - Thumbnail images
    - Subtitle files (SRT, VTT)
    
    **Download sections** - Download specific time ranges instead of the entire media
    
    **Output format** - Convert to MP4, MKV, or WebM, or recode video quality
    
    For your first download, you can skip these options and use the defaults.
  </Step>

  <Step title="Choose download location">
    By default, VidArchive downloads to your system's Downloads folder. To change this:
    
    1. Click the folder icon next to the download path
    2. Select a different folder from the file picker
    3. Optionally, enable **Remember this folder** in Settings to use it for future downloads
  </Step>

  <Step title="Start the download">
    Click the **Download** button to start downloading your media file.
    
    You'll be automatically redirected to the **Downloads** page where you can:
    - Monitor real-time progress with a progress bar and percentage
    - See download speed and estimated time remaining
    - View the yt-dlp command output in real-time
    
    <Warning>
      Don't close VidArchive while downloads are in progress. You can minimize it, but closing will cancel active downloads.
    </Warning>
  </Step>

  <Step title="Access your downloaded media">
    When the download completes:
    
    - The progress bar turns green and shows "100%"
    - The status changes to "Completed"
    - You can click **Show in folder** to open the download location
    - Or click **Play media** to open the file in your default media player
    
    Your download is also saved to **Download history** where you can access it anytime.
  </Step>
</Steps>

## Common questions

<Accordion title="Can I download multiple files at once?">
  Yes! VidArchive supports concurrent downloads. By default, it allows 2 simultaneous downloads, but you can adjust this in **Settings** up to 5 concurrent downloads.
  
  Any additional downloads will be queued and start automatically when a slot becomes available.
</Accordion>

<Accordion title="Where are my downloaded files saved?">
  By default, VidArchive saves files to your system's Downloads folder. You can:
  - Change the location for each download using the folder picker
  - Set a default folder in **Settings** → **Remember previous downloads folder**
  - Use custom filename templates in **Settings** (advanced)
</Accordion>

<Accordion title="What if the download fails?">
  If a download fails:
  1. Check the error message in the download output
  2. Common issues include network problems, unavailable media, or format issues
  3. Click **Retry** to attempt the download again
  4. If it continues to fail, see the [troubleshooting guide](/advanced/troubleshooting)
</Accordion>

<Accordion title="Can I pause and resume downloads?">
  Yes! Use the **Pause** button on any active download to pause it. Click **Resume** to continue from where it left off. You can also pause all downloads at once using **Pause all** in the Downloads page.
</Accordion>
