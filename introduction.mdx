VidArchive is a powerful desktop application that makes downloading and managing media from the web simple and intuitive. Built with Electron and React, it provides a modern interface for downloading videos and audio with full control over formats, quality, and download options.

## Get started in minutes

VidArchive handles all the complexity of media extraction behind the scenes using yt-dlp and ffmpeg, while giving you a clean, user-friendly interface to work with.

## Key features

**Interactive format selection**
View all available formats for any media URL and choose exactly what you want - resolution, codec, file size, and more.

**Rich download options**
Embed thumbnails, chapters, subtitles, and metadata directly into your media files. Download specific sections, write additional files, and customize output formats.

**Download management**
Track running downloads with real-time progress, view your complete download history, and search through past downloads.

**URL history**
Keep track of every URL you've fetched, making it easy to revisit and re-download media with different settings.

## What you can download

VidArchive supports any media source that yt-dlp supports, including:

- Video platforms (YouTube, Vimeo, Twitch, and hundreds more)
- Audio platforms
- Social media posts
- Live streams (with support for downloading from the start)
- And many more sources

<Note>
  VidArchive automatically validates URLs and lets you know if a source is supported before you attempt to download.
</Note>
